# 3D 서바이벌 게임
플레이어가 자원을 수집하고 생존을 위해 적과 싸우는 게임


## 🖥️ 프로젝트 소개
스파르타 코딩클럽 내일배움캠프 "유니티 숙련 주차"의 팀 프로젝트 입니다.
러스트를 참고하여 제작했습니다.
